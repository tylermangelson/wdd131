Overview

Purpose

The purpose of TMF Agents is to provide a simple library of pre-made AI agents in the form of task markdown files. They consist of thoroughly written instructions that can be given to an AI chatbot to perform roles such as a research assistant, a fact checker, a newsletter generator, and more.

Audience

This site is for anyone who wants to utilize AI for chatbots more effectively in their daily lives, beyond a glorified search engine.

Dynamic elements

  • Agent Grid: The container where JS will inject dynamic agent cards.
  • Search Input: Listens for user typing to trigger the .filter() method.
  • Category Buttons: Filters the array based on agent types (e.g., "Coding").
  • Counter: A span or div updated using array.length to show total results.
  • No Results Message: A hidden element shown via if/else when no agents match.
  • Dynamic Badges: Logic-driven labels (e.g., "Beta") added to specific cards.
  • Download/View Buttons: Event-driven elements for interacting with specific objects.
